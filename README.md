# Python-Supermarket-Sales-Data-Analysis-project
Cleaning a messy supermarket sales dataset using python and python related libraries like numpy, pandas and seaborn to create visualizations to solve real world business problems.

## Project Overview

This project focused on cleaning, analyzing, and visualizing a retail sales dataset using Python. The dataset contained customer, product, transaction, and financial information but included several data quality issues such as missing values, inconsistent formats, invalid entries, and outliers.

The objective was to transform the raw data into a clean, analysis-ready dataset and generate meaningful business insights that support data-driven decision-making.

---

## Objectives

The project aimed to:

- Assess and improve data quality.
- Clean and preprocess the dataset.
- Perform exploratory data analysis (EDA).
- Identify trends, patterns, and relationships within the data.
- Answer key business questions related to sales, profitability, customer behavior, and pricing strategies.
- Present findings through effective visualizations.

---

## Data Cleaning

The following data preparation tasks were completed:

- Handled missing values.
- Corrected inconsistent categorical values.
- Converted data types.
- Removed invalid entries and treated outliers.
- Standardized text fields.
- Created new analytical features including Month, Year, Age Group, Gross Sales, Discount Amount, and Profit Margin.

---

## Business Questions Addressed

The analysis answered the following key business questions:

1. Which product categories generate the highest revenue and profit?
2. How do discounts affect revenue and profitability?
3. Which customer age groups generate the highest sales?
4. Which cities and states contribute the most revenue?
5. Which products are the most profitable?
6. Is there a relationship between unit price and purchase quantity?
7. Which payment methods are associated with higher-value purchases?
8. How do sales and profits change over time?
9. Which financial variables have the greatest impact on profitability?
10. Which customer segments generate the highest Average Order Value (AOV)?
